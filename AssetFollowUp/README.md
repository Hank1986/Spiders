# 资产跟进抓取助手

合规说明：脚本仅辅助人工登录，不包含验证码破解。使用前请确认已获得目标站点访问与抓取授权，遵守服务条款与相关法律法规。

## 目录
- `src/index.js` 单页示例：登录并抓取当前页数据
- `src/export.js` 扩展：自动尝试翻页，导出全部页 CSV/JSON
- `login_page.png` 登录截图（运行时生成）

## 安装
```bash
cd Spiders/AssetFollowUp
npm install
```

## 运行示例（单页）
```bash
npm start
```
查看生成的 `asset_followup_page1.json`。

## 批量导出
```bash
npm run export
```
生成 `asset_followup_all.csv` 与 `asset_followup_all.json`。

## 使用流程
1. 运行脚本后查看生成的 `login_page.png` 手动读取验证码。
2. 在终端输入验证码回车。
3. 若脚本无法自动定位菜单或按钮，手动在浏览器中完成，再继续。
4. 数据抓取完成后文件写入当前工作目录。

## 定制选择器
根据实际页面结构修改：
- 登录输入框选择器
- 登录按钮选择器
- 表格与分页选择器

## 注意事项
- 不进行验证码绕过或破解。
- 控制访问频率，避免对目标站造成负载压力。
- 若页面使用接口分页，可抓包识别合法接口再行开发（需授权）。

示例脚本，供内部合规使用。
